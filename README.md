## ✨ Key Features

### 🧠 AI & Code Intelligence
- **Code Completion**: Smart context-aware code suggestions powered by LLMs
- **Code Generation**: Auto-generate entire functions, components, and files from natural language
- **Code Review**: Get detailed analysis, suggestions, and improvements for your code
- **Refactoring**: Automated intelligent refactoring and code cleanup
- **Documentation**: Generate professional documentation for your codebase

### 💻 Integrated Development Environment
- **Multi-Language Support**: JavaScript, TypeScript, Python, Go, Rust, SQL, HTML, CSS, Markdown
- **Code Editor**: Built on CodeMirror with syntax highlighting, themes, and keyboard shortcuts
- **Terminal Emulator**: Integrated terminal with full shell access
- **File System**: Full file management and tree browser
- **Git Integration**: Visual git operations, commits, and diff viewer

### 🎨 Modern UI/UX
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Dark & Light Themes**: Choose your preferred working environment
- **Rich Component Library**: Beautiful, accessible UI components from Radix and shadcn
- **Animations**: Smooth transitions and micro-interactions for a polished experience
- **Keyboard-First**: Full keyboard accessibility and shortcuts

### 🔧 Developer Tools
- **Real-Time Sync**: Instant collaboration with team members
- **Hot Reload**: Fast refresh during development
- **Type Safety**: Full TypeScript support for reliability
- **Build System**: Optimized Vite-based tooling
- **API Server**: Robust backend with Express and Drizzle ORM

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm/pnpm/yarn

### Installation
```bash
# Clone the repo
git clone https://github.com/Hayredin950/Hayre-IDE.git
cd Hayre-IDE

# Install dependencies (for both frontend and backend)
cd hayre-ide/artifacts/hayre-cli && npm install
cd ../api-server && npm install

# Start backend
npm run dev

# In a new terminal, start frontend
cd ../hayre-cli
npm run dev
```

### Usage
Open your browser and go to http://localhost:5173 - Hayre IDE is ready to use!
